数据结构：
一个用户id生成器，一个所有用户的集合
一个article的id生成器，article的hash，其键为article的id
一个article的有序集合，分值为时间戳
一个article的有序集合，分值为投票得分
若干个article的投票用户集合，防止用户重复投票
若干个群租的集合

方法：
1.新增用户，并返回用户id
2.新增article散列，键为article的id，field为属性名，value为属性值
新增article的投票用户集合，
article的时间戳集合中添加一个记录，article的得分集合中增加一个记录
3.投票，判断文章的发布日期是否在一周内，判断用户是否已经投过票，
条件合适，更新article得分集合的记录，article散列的投票数增长1，article的投票用户集合增加一个记录
4.按分值或最新发布时间，显示文章列表
5.根据文章id显示文章详情
6.创建分组，将文章添加到分组中
7.显示分组排序文章，并缓存结果60秒
